# Múltiplo de 7

Chegou a hora de testar seus dotes de programação, escreva um algoritmo que identifique múltiplos de 7.

## Entrada

Como entrada terá um inteiro N onde N < 1000

## Saída

Informe se é múltiplo de 7

## Exemplo

Entrada Saída
70

É MULTIPLO DE 7

13

ESSE NÃO É

## Exemplos

| Entrada | Saída           |
| ------- | --------------- |
| 70      | É MULTIPLO DE 7 |
| 13      | ESSE NÃO É      |
