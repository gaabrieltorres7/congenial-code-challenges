# Pares de 1 a 100

Faça um programa que mostre os números pares entre 1 e 100, inclusive.

# Entrada

Neste problema não há entrada.

# Saída

Imprima todos os números pares entre 1 e 100, inclusive se for o caso, um em cada linha.

## Exemplos

| Saída |
| ----- |
| 2     |
| 4     |
| 6     |
| 8     |
| ...   |
| 100   |
